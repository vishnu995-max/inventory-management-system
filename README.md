# Inventory Management System (DBMS Project)

A web-based Inventory Management System designed as part of a Database Management Systems academic project.

## Project Overview
This system demonstrates relational database design and implementation for managing:
- Products
- Suppliers
- Warehouses
- Orders
- Payments
- Shipments

## Database Concepts Implemented
- ER Model Design
- Relational Schema Design
- Primary & Foreign Key Constraints
- SQL Queries (Simple & Complex)
- Joins and Subqueries
- Set Operations
- Views
- Triggers
- Cursor Procedures
- Database Normalization
- Optimistic Concurrency Control

## Technologies Used
- SQL
- SQLite
- Flask (Conceptual Backend Design)
- HTML, CSS (Frontend Concept)
